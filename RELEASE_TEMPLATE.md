## Installation

1. Open Obsidian
2. Go to Settings â†’ Community Plugins
3. Disable safe mode if prompted
4. Click Browse community plugins
5. Search for "DeepSeek AI Assistant"
6. Click Install
7. Once installed, enable the plugin
8. Configure your DeepSeek API Key in the plugin settings

## Configuration

After installation, you'll need to configure the plugin:

1. Obtain a DeepSeek API key from [platform.deepseek.com](https://platform.deepseek.com)
2. In Obsidian, go to Settings â†’ DeepSeek AI Assistant
3. Paste your API key in the designated field
4. Adjust other settings as needed (model, temperature, etc.)

## Usage

The plugin offers multiple ways to interact with the AI:

- **Command Palette**: Press `Ctrl/Cmd + P` and search for "DeepSeek"
- **Right-click Menu**: Select text and right-click to access analysis options
- **Chat Dialog**: Open the chat interface for conversational interactions
- **Sidebar Panel**: Access AI tools through the sidebar
