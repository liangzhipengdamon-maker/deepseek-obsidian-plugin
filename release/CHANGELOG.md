# Changelog

## [1.0.0] - 2025-12-15

### Added
- Initial release of DeepSeek Obsidian Plugin
- AI-powered chat interface with streaming responses and reasoning process visualization
- Knowledge base search functionality
- Note analysis, summarization, and enhancement capabilities
- Multiple integration points: command palette, right-click menu, sidebar panel, and chat dialog
- Persistent conversation history across Obsidian restarts
- Configurable API settings including API key, model selection, and temperature

### Features
- **AI Chat Interface**: Copilot-style multi-turn conversation with real-time streaming responses
- **Knowledge Base Integration**: Automatic context retrieval from your Obsidian notes
- **Note Processing**: Analyze, summarize, and enhance your note content
- **Multiple Access Methods**: Various ways to interact with the AI assistant
- **Privacy Focused**: Read-only access to your knowledge base with no data uploaded
- **Persistent Memory**: Conversation history preserved across sessions

### Technical Details
- Built with TypeScript and Obsidian API
- Uses DeepSeek's advanced reasoning models
- Implements streaming responses for better user experience
- Supports both regular and reasoning mode responses
- Local caching for conversation persistence
